# Changelog

All notable changes to MediaTube will be documented in this file.

## [1.0.0] - 2026-02-04

### Added
- In-app browser with media detection
- YouTube video/audio stream extraction
- Multi-format download support (video, audio, MP3)
- Background download with notifications
- Download queue with pause/resume
- Download history
- Auto-update checker

### Features
- Supports YouTube, Twitter/X, Instagram, and other platforms
- Quality selection (up to 4K)
- Audio-only MP3 extraction
- FFmpeg-based video/audio merging

### Technical
- Flutter 3.x with provider state management
- ABI split builds for smaller APK sizes
- Comprehensive proguard rules for release builds
